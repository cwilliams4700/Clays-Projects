Guardia NPM Scanner:

Proactive Scanning: Uses the OSV (Open Source Vulnerabilities) database to find known security risks for any npm package and version.
AI-Powered Insights: Integrates Gemini 2.0 Flash with Google Search Grounding to analyze vulnerabilities, provide remediation steps, and find the latest security advisories that might not yet be in official databases.
Polished Technical UI: A high-density, professional dashboard built with shadcn/ui, Tailwind CSS, and Framer Motion for smooth interactions.
Scan History: Persists your recent scans locally so you can quickly revisit previous security checks.
Detailed Vulnerability Reports: Displays severity levels (Low to Critical), fixed versions, and direct links to security references.
